English:

This is a program for downloading video or audio (with conversion to mp3) from popular video hosting sites such as youtube.com and others. It uses the yt-dlp library and the FFmpeg program to function.
To get started:

1) Install FFmpeg for your system. It can be downloaded from https://ffmpeg.org/download.html;
2) Download and extract the contents of the YouTube-Downloader-Portable.zip archive from this repository into any folder;
3) Run the file download_deps.bat from the same folder where you extracted the archive from step 2. It will create a /bin folder;
4) Launch the yt_downloader.exe file, this will open a command prompt;
5) Follow the prompts in the command prompt and answer the questions. The downloaded files will be saved in the same folder as the program itself.
